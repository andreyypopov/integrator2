Detailed Description

Class for storage of mesh information.

Main functionality:

  • import of mesh data (vertices and cells) from the DAT file;
  • mesh processing (calculation of normals, measures, centers);
  • determining cell neighbors.
Remarks
All the data is stored in the GPU memory

◆ loadMeshFromFile()

bool Mesh3D::loadMeshFromFile ( const std::string & filename,
double scale = 1.0 )

Import mesh from the DAT file.

Parameters
filenameName of the .dat file
scaleScaling parameter for vertex coordinates
Returns
true Mesh successfully loaded and transferred to the GPU memory
false Error while loading the mesh

Mesh file is supposed to contain the following data:

  1. Number of vertices, number of cells (triagnles).
  2. Vertex data in the form of: index, x, y, z.
  3. Triangle data in the form of: 203, index1, index2, index3 (203 is the triangle element type, vertices are referenced by their indices, base-1).

Vertex and cell data are immediately transferred to the GPU vectors.

◆ prepareMesh()

void Mesh3D::prepareMesh ( )

Unified function with calls for all preparatory functions for the mesh.

Includes calculation of cell normals, centers and measures and determining types of neighbors for all cell pairs. All procedures are carried out on GPU.
